# SQL でテーブルを作成する（DDL 作成）

## 目的

これまで設計した内容（テーブル・カラム・リレーション・制約）を  
**実際の SQL として定義する段階**です。

この SQL は **DDL（Data Definition Language）** と呼ばれます。

---

## 使用するソフトウェア

### DB 本体

一般的には次のどれかを使います。

- MySQL
- PostgreSQL
- SQLite
- Oracle
- SQL Server

### DB クライアントツール

PC では次のようなツールを使うことが多いです。

- **DBeaver（おすすめ）**
- MySQL Workbench
- TablePlus
- pgAdmin

これらのツールの **SQL エディタ** に SQL を書きます。
